# 架空サイト制作「名無し乗馬クラブ」

架空サイト制作チャレンジ第三弾。  
React + Vite で構築した架空の乗馬クラブサイトです。

## リンク
https://upotsu.github.io/Nanashi_Horse_Club/

## なぜこのサイトを作ったのか
・昔、縁のあった乗馬クラブのサイトを作ってみたかったので予約サイトのポートフォリオ用に制作。  
・前回は1ページのサイトだったので今回は新しくReact+Viteで複数ページのサイト構築に挑戦してみた。  

## ページ構成
- ホーム
- 乗馬レッスン案内
- 牧場カフェ
- 会社概要
- お問い合わせ

## 特徴
- HashRouter による複数ページ構成
- 写真と動画を活かしたヒーローセクション
- レッスン案内ページにカレンダー予約UIを実装
- アップロード素材を各ページへ組み込み済み
